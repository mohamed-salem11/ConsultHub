# ConsultHub

ConsultHub is a comprehensive online consultation platform that connects clients with certified consultants in a secure, organized, and user-friendly digital environment.  
The platform enables professional booking, management, and evaluation of consultations.

---

##  Key Features

###  Client Experience
- Browse consultations by categories.
- View full details of each consultation (description, price, rating, consultant).
- Book and pay securely online.
- Track booking status.
- Rate consultations after completion.

###  Consultant Experience
- Create and manage a professional profile.
- Add new consultations.
- Manage incoming bookings.
- Monitor reviews and build professional reputation.

###  Admin Dashboard
- Manage consultations and categories.
- Review and approve/reject consultations to ensure quality.
- Handle requests to join as a consultant.
- Track reports and issues.
- Delete or edit inappropriate content.

###  Quality & Review System
- Consultation lifecycle:
  - Draft – initial stage
  - Pending Review – awaiting admin approval
  - Approved / Rejected – final decision
- Ensures high-quality content and user trust.

###  Ratings & Trust
- Star-based rating system for consultations.
- Ability to edit or remove ratings.
- Clients can request post-session reviews to ensure credibility.


---

##  Tech Stack
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Identity Framework (Roles & Authentication)
- Bootstrap & Razor Views
- Stripe API (secure payments)
